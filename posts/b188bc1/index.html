部署应用前的一些工作

  • 下载苹果CMS源码

    • 网址链接

  • 下载主题

    • 网址链接

安装和配置步骤

1. 添加域名和设置DNS

  • 登入Serv00面板添加自己的域名点击 WWW websites 然后点击Add new websites
  • 注意添加自己的域名并关闭DNS。如果使用 Serv00 域名，需开启全局。

2. 创建一个MySQL

  • 进入控制面板，创建一个新的 MySQL 数据库。

3. 进入WWW网站权限

  • 进入 WWW websites，设置相应的权限。

4. 开始上传前面下载的CMS并解压

  • 使用 File manager 进入文件管理，删除指定文件夹中的所有文件。
  • 上传下载的 CMS 压缩包并解压。

5. 解压到domains–你的域名–public html文件下。然后删掉zip压缩包

  • 然后鼠标右击文件，点击Explore，然后右击出现的文件，点击Extract开始解压到domains–你的域名–public html文件下。然后删掉zip压缩包

6. 上传并解压主题压缩包

  • 进入 maccms 文件夹，上传并解压主题压缩包到 template 目录下。
  • 将主题文件夹改名为 mxone 并删除压缩包。

7. 然后进入template文件夹下

  • 上传mxone压缩包，解压到template目录下，将主题文件改名为mxone，并删除zip文件

8. 移动maccms文件

  • 将maccms的所有文件到public html文件下，并删除maccms10-2024.1000.4010空文件


9. 在public html文件下找到admin.php文件

  • 找到后将其改名，我以admin123.php为例（可自行修改，这是后台的后缀）

10. 更改PHP版本为74（这部及其重要，不然后面主题无法获取）

  • 进入domains–域名文件下，添加一个文档：- .htaccess
  • 点击:heavy_plus_sign:–New empty file，
  • 右击文档 .htaccess，依次点击：View/Edit –Choose other–Text Editor
  • 然后将代码放入文件中，点击左上角save保存

11. 访问你的域名安装后台 看到此图片代表已经成功 然后点击同意并安装

12. PHP版本一定要对 - PHP版本7.4

13. 管理员账号密码，点击安装

  • 数据库名称账号和密码就是前面MySQL申请的。对应填入即可，然后点击测试数据库连接。出现数据库连接成功**就没问题了。

14. 先进入前台：打开一个新网页，输入你的域名 比如 https://jinx.200038.xyz

  • 打开一个新网页，进入系统管理后台，你的域名/admin123.php（后缀前面如果不一样进行对应调整）

15. 输入后台账号和密码 然后点击升级

16. 配置主题

  • 点击自定义菜单配置，复制下面的主题文字：（admin123.php前面改了对应调整），然后并保存

17. 然后点击系统，网站参数配置 把网站模板和手机模板改为mxone，并保存

18. 然后新开一个网页打开你的域名 比如 https://jinx.200038.xyz 变下图就可以了。

19. 最后，刷新你的后台 - 点击mxoneX主题并保存。（后面可以自行调整里面配置）

20. 无法访问以及运行内存超载导致的网站卡顿的解决方法

首先用SSH连接上serv00上的服务器，在邮件里有SSH的地址，然后填写账号密码，我用的软件是FinalShell，进去之后，输入以下命令, yourname替换为你的用户名

21. 采集片源的话建议大家看视频,这里不方便添加图片了。
